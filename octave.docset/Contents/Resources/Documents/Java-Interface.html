A.4 Java Interface

The Java Interface is designed for calling Java functions from within Octave. If you want to do the reverse, and call Octave from within Java, try a library like javaOctave (https://kenai.com/projects/javaoctave/pages/Home) or joPas (http://jopas.sourceforge.net/).
